MapLead — a five-stage ingestion pipeline behind an API

Turns an unstructured public source into a scored, schema'd dataset. Playwright scrapes Google Maps, an enrichment stage probes each business site for tracking pixels, a transparent 0–100 heuristic scores it, and the run lands as a spreadsheet. Async by design: a run takes about five minutes on free-tier hardware, so the API accepts the job and delivers by email rather than holding the request open.
